## Office Printer 9K — Audio attribution (Milestone A)

This folder is expected to contain the game’s sound effects.

### Recommended free sources (pick a set and record it here)
- **Freesound (CC0/CC-BY varies per file)**: check the license on each download.
  - Example printer/mechanical recordings:
    - `https://freesound.org/people/soundandmelodies/sounds/776195/`
    - `https://freesound.org/people/soundandmelodies/sounds/776196/`
  - Example UI/error beeps:
    - `https://freesound.org/people/pryanic/sounds/780800/`
    - `https://www.freesound.org/people/SamsterBirdies/sounds/581602/`
- **itch.io UI packs** (license varies per pack):
  - `https://bleeoop.itch.io/ui-clicks`

### Files expected by the build
Drop audio files here using the stable names below. Prefer `.ogg` (or use `.mp3` and update `src/game/audio/audioKeys.js`).

- `hum_loop.ogg`
- `ui_click.ogg`
- `ui_confirm.ogg`
- `ui_error.ogg`
- `paper_feed.ogg`
- `beep_warning.ogg`
- `reboot.ogg`
- `jam_alarm.ogg`
- `ending_win.ogg`
- `ending_fail.ogg`
